# 绘图工具演示脚本 - Demo Script

## 🎬 30秒快速演示

### 场景 1: 绘制趋势线标记支撑位
**时长**: 10秒

**旁白**:
> "使用趋势线工具，轻松标记股票的支撑位和压力位。"

**操作步骤**:
```
1. 点击左侧工具栏 LINES → TRENDLINE
2. 在K线低点处点击第一个点
3. 拖动到另一个低点，点击第二个点
4. 完成！一条精确的趋势支撑线已绘制完成
```

**画面要点**:
- 🎯 工具按钮高亮（蓝色）
- 🖱️ 鼠标点击时有视觉反馈
- ✨ 绘制完成后线条发光效果
- 📊 趋势线紧贴K线低点

---

### 场景 2: 水平线标记关键价格
**时长**: 8秒

**旁白**:
> "一键绘制水平支撑/阻力线，自动显示价格标签。"

**操作步骤**:
```
1. 点击 LINES → HLINE
2. 在关键价格位置点击一次
3. 完成！横跨整个图表的水平线立即出现
```

**画面要点**:
- ⚡ 一次点击即完成
- 💰 左侧自动显示价格（如 "123.45"）
- 🌐 横跨整个图表宽度
- 🎨 使用专业天蓝色 (#0ea5e9)

### 场景 3: 选择和编辑图形
**时长**: 12秒

**旁白**:
> "选择工具让你可以随时调整绘图对象的位置和大小。"

**操作步骤**:
```
1. 按 ESC 键切换到选择工具
2. 点击之前绘制的趋势线（出现发光效果）
3. 拖动线条移动到新位置
4. 拖动端点的小圆点调整角度
5. 按 Delete 键删除不需要的图形
```

**画面要点**:
- 🔆 选中时的发光效果（shadowBlur）
- ⚪ 端点的控制点（白色实心圆）
- 🖱️ 拖动时实时跟随鼠标
- 🗑️ 删除后平滑消失

## 🎬 拍摄技巧

### 录屏设置
- **分辨率**: 1920x1080 (Full HD)
- **帧率**: 60fps (流畅度优先)
- **码率**: 10Mbps+
- **格式**: MP4 (H.264)

### 鼠标设置
- 启用鼠标点击视觉效果
- 鼠标指针大小适中（不要太小看不见）
- 考虑使用鼠标高亮工具

### 画面布局
```
┌─────────────────────────────────┐
│  Title (0-5秒)                  │
├─────────────────────────────────┤
│                                 │
│      主要演示区域                │
│      (图表 + 工具栏)             │
│                                 │
├─────────────────────────────────┤
│  Tips / 快捷键提示 (底部)        │
└─────────────────────────────────┘
```

### 节奏控制
- 每个操作停顿 1-2 秒，让观众看清
- 重要步骤可以放慢 0.5x 播放
- 转场使用淡入淡出（1秒）

### 配音要点
- 语速适中（不要太快）
- 吐字清晰
- 音量适中（背景音乐不要太响）
- 专业但不生硬
