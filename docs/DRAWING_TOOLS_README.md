## 🔧 快速参考

### 工具清单
| 工具 | 点击次数 | 用途 |
|------|----------|------|
| 趋势线 (Trendline) | 2次 | 标记趋势方向 |
| 水平线 (HLine) | 1次 | 支撑/阻力位 |
| 垂直线 (VLine) | 1次 | 时间标记 |
| 射线 (Ray) | 2次 | 长期趋势 |
| 矩形 (Rectangle) | 2次 | 区域框选 |
| 箭头 (Arrow) | 2次 | 方向标注 |
| 文本 (Text) | 1次 | 文字注释 |
| 斐波那契 (Fibonacci) | 2次 | 回调位分析 |

### 快捷键
```
ESC            - 切换到选择工具 / 取消操作
Delete/Backsp  - 删除选中对象
Ctrl+Z         - 撤销
Ctrl+Y         - 重做
Ctrl+Shift+V   - 调试面板
```

### 调试命令
在浏览器控制台输入：
```javascript
toggleChartDebug()  // 切换调试日志
toggleChartPerf()   // 切换性能监控
```
