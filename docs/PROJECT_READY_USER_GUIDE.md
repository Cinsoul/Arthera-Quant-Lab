## 📊 核心功能使用流程

### 🔍 选股器 (StockPicker)
1. **搜索股票**: 在搜索框输入股票代码或名称
2. **查看实时价格**: 显示当前价格、涨跌幅、成交量
3. **支持的股票类型**:

### 📈 图表工作台 (Chart Workbench)
1. **进入方式**: 点击股票列表中的绿色图表按钮
2. **功能特性**:
   - 专业K线图显示
   - 多时间周期切换 (1D, 5D, 1M, 3M, 6M, 1Y)
   - 技术指标展示 (MA5, MA10, MA20)
   - 成交量分析
   - 实时数据更新

### 🖥️ 全屏图表 (FullScreen Mode)
1. **进入方式**: 在Chart Workbench点击"完整图表"按钮
2. **专业级功能**:
   - Bloomberg级图表交互
   - 高级技术指标 (RSI, MACD, KDJ)
   - 专业画线工具

## 🎯 技术架构

### 📱 前端技术栈
- **框架**: React 18 + TypeScript
- **构建工具**: Vite
- **图表库**: 自研专业图表系统
- **UI组件**: Tailwind CSS + Lucide Icons
- **状态管理**: React Hooks + Context

### 🔧 后端技术栈
- **框架**: FastAPI + Python
- **数据源**: 多API聚合 (Finnhub, FMP, TwelveData等)
- **API设计**: RESTful统一接口
- **数据格式**: 标准化OHLCV格式

### 🔗 核心API端点
```bash
# 实时行情
GET /api/unified/quote/{symbol}

# 历史数据  
GET /api/unified/historical/{symbol}?period=daily&days_back=30

# 健康检查
GET /health

# API密钥验证
GET /api/v1/market/test/{provider}?apiKey={key}
```
