# 📊 MA均线指标使用指南

## 快速开始

### 什么是MA均线？

MA（Moving Average）移动平均线是技术分析中最常用的指标之一，用于平滑价格波动，识别趋势方向。

**我们提供三条MA均线**:
- **MA5** (橙色) - 5日移动平均线，短期趋势
- **MA10** (绿色) - 10日移动平均线，中期趋势  
- **MA20** (紫色) - 20日移动平均线，长期趋势

---

## 如何使用

### 1. 打开图表工作台

**路径**: 主导航 → **Chart Workbench**

### 2. 查看MA均线

MA均线**默认已开启**，你会在图表上看到：
- 橙色线（MA5）
- 绿色线（MA10）
- 紫色线（MA20）
- 左上角的图例标注

### 3. 切换MA显示

点击工具栏的 **MA** 按钮：
- **橙色背景** = MA开启
- **灰色背景** = MA关闭

```
┌────────────────────────────────────┐
│ 工具栏                             │
├────────────────────────────────────┤
│ [K线] [线图] [面积] | [MA] [IND...] │
│                       ↑            │
│                    点击这里         │
└────────────────────────────────────┘
```

### 4. 切换股票

使用下拉菜单选择不同股票：
- 600519 - 贵州茅台
- 300750 - 宁德时代
- 000858 - 五粮液
- 600036 - 招商银行

MA均线会**自动重新计算**。

### 5. 切换时间周期

点击周期按钮：
- **1D** - 1天
- **5D** - 5天
- **1M** - 1个月
- **3M** - 3个月
- **6M** - 6个月
- **1Y** - 1年
- **YTD** - 年初至今

MA均线会**自动适应新周期**。

### 趋势判断

**上升趋势**:
- ✅ 价格在MA均线上方
- ✅ MA5 > MA10 > MA20（多头排列）
- ✅ 所有MA均线向上

**下降趋势**:
- ⚠️ 价格在MA均线下方
- ⚠️ MA5 < MA10 < MA20（空头排列）
- ⚠️ 所有MA均线向下

**震荡行情**:
- ➡️ 价格来回穿越MA均线
- ➡️ MA均线相互缠绕
- ➡️ 方向不明确

### 交易信号

**买入信号**:
1. **金叉**: MA5上穿MA10或MA20
2. **突破**: 价格突破MA均线压力位
3. **支撑**: 价格回踩MA均线获得支撑

**卖出信号**:
1. **死叉**: MA5下穿MA10或MA20
2. **破位**: 价格跌破MA均线支撑位
3. **压力**: 价格反弹遇到MA均线压力

## 高级技巧

### 1. 多周期共振

同时查看不同周期的MA均线：
1. 打开一个图表查看 **1M**（短期）
2. 点击全屏按钮
3. 切换到 **3M** 或 **6M**（中期）
4. 对比两个周期的MA排列

**如果多个周期都显示多头排列 → 强烈上升趋势**

### 2. MA距离判断

观察价格与MA均线的距离：
- **过度远离**: 可能面临回调
- **紧贴均线**: 支撑/压力强
- **频繁穿越**: 震荡行情

### 3. MA斜率分析

观察MA均线的倾斜角度：
- **陡峭向上**: 趋势强劲
- **平缓**: 趋势减弱
- **水平**: 无趋势

### 4. 成交量配合

结合下方的成交量柱：
- **突破+放量**: 信号可靠
- **突破+缩量**: 信号存疑
- **跌破+放量**: 警惕风险

## 常见问题

### Q1: 为什么MA均线有时不显示？

**A**: 检查MA按钮状态：
- 橙色背景 = 已开启 ✅
- 灰色背景 = 已关闭 ❌

**解决**: 点击MA按钮开启

---

### Q2: MA均线在图表起始处断开？

**A**: 这是正常现象。MA5需要5个数据点才能计算，MA20需要20个。

**说明**:
- MA5从第5根K线开始显示
- MA10从第10根K线开始显示
- MA20从第20根K线开始显示

---

### Q3: 切换股票后MA均线没变化？

**A**: 应该会自动更新。如果没有：

**解决步骤**:
1. 刷新浏览器页面
2. 重新选择股票
3. 检查浏览器Console是否有错误

---
